← 건다 홈

개인정보처리방침

최종 수정일: 2026년 4월 16일

건다(이하 "앱")는 이용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다.

1. 수집하는 정보

  • 닉네임 (사용자가 직접 입력)
  • 앱 사용 시간 데이터 (UsageStatsManager — 기기 내에서만 처리)
  • 걸음수·수면 데이터 (Health Connect — 기기 내에서만 처리)
  • 집행자 연락처 (사용자가 직접 입력, 기기 내에서만 저장)

2. 정보의 이용 목적

  • 서약 조건의 자동 검증
  • 위반 시 집행자 알림 발송
  • 앱 서비스 제공 및 개선

3. 제3자 제공

건다는 이용자의 개인정보를 외부에 판매하거나 제공하지 않습니다. 카카오페이 결제 연동 시 최소한의 거래 정보가 카카오 정책에 따라 처리됩니다.

4. 보유 및 파기

모든 데이터는 기기 내 로컬 DB에만 저장됩니다. 앱 삭제 시 모든 데이터가 함께 삭제됩니다.
